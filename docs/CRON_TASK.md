# 定时任务功能文档

> OA 智审平台 — 定时任务模块前端功能规范

---

## 1. 模块概述

定时任务模块包含三种任务类型，核心设计理念是：定时任务本身不直接使用 AI，而是通过定时触发审核工作台的能力来完成工作。

| 任务类型 | 用途 | 是否涉及邮件推送 | 数据来源 |
|---------|------|----------------|---------|
| 批量审核 | 定时执行审核工作台的待办 AI 审批 | 否 | 审核工作台待办列表 |
| 日报推送 | 定时获取当日审核数据，组装邮件推送 | 是 | 审核工作台当日审核结果 |
| 周报推送 | 定时获取本周审核数据，组装邮件推送 | 是 | 审核工作台本周审核结果 |

---

## 2. 前台业务端：定时任务中心（`/cron`）

### 2.1 任务卡片列表

以卡片网格形式展示所有定时任务，每张卡片包含：
- 任务类型标签（批量审核/日报推送/周报推送）
- 运行状态（运行中/已暂停）
- Cron 表达式及中文描述
- 推送邮箱（仅日报/周报类型显示）
- 执行统计（成功/失败次数、上次执行时间）
- 操作按钮（立即执行、暂停/启用、编辑、复制、删除）

### 2.2 新建定时任务

弹窗表单包含：
- 任务类型选择（选择后显示对应的功能说明）
  - 批量审核：说明"通过定时任务自动执行审核工作台的待办 AI 审批，不涉及邮件推送"
  - 日报推送：说明"通过定时任务执行审核工作台获取当日审核数据，组装邮件后推送日报"
  - 周报推送：说明"通过定时任务执行审核工作台获取本周审核数据，组装邮件后推送周报"
- 执行计划（预设选项 + 自定义 Cron 表达式构建器）
- 下次执行时间预览
- 推送邮箱（仅日报/周报类型显示此字段）

### 2.3 编辑定时任务

与新建类似，推送邮箱字段同样仅在日报/周报类型时显示。

## 3. 后台管理端：规则配置 — 定时任务配置

租户管理员在"规则配置"页面的"定时任务配置"页签中管理三种任务类型的配置。

### 3.1 批量审核配置

仅包含一个配置项：
- 启用/禁用开关
- 单次执行上限（1-500），控制每次定时任务最多处理的待办流程数量

不包含内容模板、AI 配置、用户权限等页签（批量审核直接调用审核工作台的 AI 审批能力）。

### 3.2 日报推送 / 周报推送配置

包含：
- 启用/禁用开关
- 内容格式选择（HTML / Markdown / 纯文本）
- 内容模板编辑（邮件主题、头部、正文模板、底部）
- 变量插入功能（参考审核工作台的变量插入方式）

不包含 AI 配置页签（日报/周报通过定时任务执行审核工作台获取数据后，按模板组装邮件内容，不独立使用 AI）。
不包含用户权限页签（用户不再可以修改定时任务配置）。

### 3.3 可用变量列表

日报推送变量：

| 变量 | 说明 |
|------|------|
| `{{date}}` | 当前日期 |
| `{{time}}` | 数据截止时间 |
| `{{total}}` | 今日审核总数 |
| `{{approved}}` | 通过数量 |
| `{{rejected}}` | 驳回数量 |
| `{{revised}}` | 建议修改数量 |
| `{{pass_rate}}` | 通过率百分比 |
| `{{detail_list}}` | 审核明细列表 |
| `{{statistics}}` | 统计数据汇总 |

周报推送变量：

| 变量 | 说明 |
|------|------|
| `{{week}}` | 当前周数 |
| `{{date_range}}` | 周起止日期范围 |
| `{{time}}` | 报告生成时间 |
| `{{total}}` | 本周审核总数 |
| `{{trend}}` | 较上周增减描述 |
| `{{compliance_rate}}` | 合规率百分比 |
| `{{compliance_trend}}` | 合规率环比变化 |
| `{{detail_list}}` | 审核明细列表 |
| `{{statistics}}` | 统计数据汇总 |

## 4. 个人设置：定时任务

用户在个人设置的"定时任务"页签中仅可配置：
- 默认推送邮箱：日报/周报推送的接收邮箱

已移除的功能：
- ~~任务类型列表及分类配置~~
- ~~内容模板编辑~~
- ~~AI 配置~~
- ~~推送设置（单独邮箱覆盖、执行计划）~~

---

## 5. 与审核工作台的关系

定时任务模块与审核工作台的关系是"调用者"与"执行者"：

- 批量审核：定时触发 → 调用审核工作台的 AI 审批接口 → 逐条处理待办流程
- 日报推送：定时触发 → 调用审核工作台获取当日审核结果数据 → 按模板组装邮件 → 推送
- 周报推送：定时触发 → 调用审核工作台获取本周审核结果数据 → 按模板组装邮件 → 推送
